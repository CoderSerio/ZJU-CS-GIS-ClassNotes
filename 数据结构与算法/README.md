# 数据结构与算法课程

《数据结构基础》 2018-2019 春夏 徐镜春
《高级数据结构与算法分析》 2018-2019 春夏 王灿

- 这两门课程是同时修读的
- 由于有诚信守则的问题，作业题目和代码什么的我就没有放上来了。这边基本上都是自学的时候写的或者其他课程的时候做的杂七杂八的数据结构相关课程作业。

### 文件目录

- search_engine - 一个使用倒排索引的搜索引擎demo
- pta - pta题解，自己练习的公开题目集
- DataStructures - 一些数据结构的实现
    - AVL树
    - 二叉搜索树
    - splay树
    - 队列
    - 跳跃表
